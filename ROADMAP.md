# Goal

By the completion of this roadmap, readers will be able to:

- Understand every mathematical concept behind Transformers.
- Implement every component using only NumPy.
- Build Encoder and Decoder stacks from scratch.
- Train a miniature Transformer model.
- Understand modern Transformer variants such as GPT and BERT.
